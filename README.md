## Overview
Web-Based Synthesizer is a browser application that generates sound in real time using the Web Audio API. The project allows users to experiment with basic sound synthesis directly in the browser without requiring any external software or plugins.

This project focuses on audio programming concepts and low-level browser APIs.

---

## Features
- Real-time sound generation in the browser
- Multiple waveform types (sine, square, sawtooth, triangle)
- Adjustable frequency and volume controls
- Instant audio feedback
- Works completely client-side

---

## Tech Stack
- HTML
- CSS
- JavaScript
- Web Audio API

---

## How It Works
1. An AudioContext is created using the Web Audio API.
2. Oscillators generate waveforms at selected frequencies.
3. Gain nodes control output volume.
4. User inputs dynamically update audio parameters in real time.
